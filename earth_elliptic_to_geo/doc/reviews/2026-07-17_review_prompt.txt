Our results (machine-tight, defects ~5e-12, independently re-derived from artifacts):

- structure: 19-24 switches, 7.3-8.1 revs, burns concentrated at APOGEE (paper Table 3: 18 switches, 7.5 revs) -- matches

Transcription essentials (casadi_lt_2body.m):
- State x = [r(3); v(3); m; t; cScale] (9 rows). Control u = [alpha(3); s] (4 rows), cone-eliminated: thrust = s*Tmax*alpha/m, ||alpha||=1, s in [0,1].
- Sundman clock: dt/dtau = cScale * kappa(r), kappa = ||r||^pSund, pSund = 1.5. tau_f is FIXED (= tauf0, a constant, NOT a decision variable); cScale is a slack STATE, constant along the trajectory (tied by its own zero row in the defect equation), which lets physical t_f float or be pinned. This is Betts' sparse free-time trick: a free scalar tau_f would multiply every defect and create one dense KKT column (MUMPS fill-in / OOM at large N), so we carry the freedom in a state instead.
- Trapezoidal collocation on a UNIFORM tau-mesh (sigma = tau/tauf0 in [0,1], dtau_k = tauf0 * diff(sigma)).
- Objective (mode 'fixedtf'): J(eps) = Int[s]dt - eps*Int[s(1-s)]dt with the physical-time measure dt = cScale*kappa*dtau, swept eps 1 -> 0. eps=1 is min-energy (strictly convex in s, smooth ramp); eps=0 is min-fuel (linear in s, bang-bang).
- Mode 'mintime': s == 1 pinned, objective = minimize t(tau_f); cScale carries the free time.
- Costates are read from the KKT duals of the defect constraints: out.lamDef is [9 x N], columns = INTERVALS, rows = [lam_r(3); lam_v(3); lam_m; lam_t; lam_cScale].
