# min_fuel_papers — source papers (reference PDFs)

Reference literature for the min-fuel low-thrust campaigns. Load-bearing
items (cited throughout the campaign docs):

- `Gergaud-Haberkorn-Martinon-JournalGuidance2004-preprint.pdf` — THE
  elliptic→GEO benchmark (JGCD 27(6), 2004) reproduced by
  `../earth_elliptic_to_geo/`.
- `COCV_2001__6__239_0.pdf` — Caillau & Noailles 2001 (source of the
  benchmark constants incl. Isp = 1994.8 s).
- `2015-8-J-LowThrustMinimumFuelOptimizationRTBP.pdf` — Zhang et al. 2015
  (RTBP min-fuel; basis of the ztl indirect campaign).
- `Leomanni2021_low_thrust_orbit_transfers_made_easy.pdf` — surveyed for the
  indirect band campaign.

The rest are supporting reading (homotopy methods, low-thrust surveys,
cislunar transfer papers). The MfMax solver manuals live with the code at
`../earth_elliptic_to_geo/indirect/mfmax/mfmax_docs/`.
